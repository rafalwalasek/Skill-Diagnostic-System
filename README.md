# Skill Diagnostic System

## Opis projektu

Skill Diagnostic System to backendowy system służący do diagnozowania i śledzenia rozwoju umiejętności w czasie.

Projekt powstał, ponieważ brakowało mi uporządkowanego sposobu mierzenia postępów w nauce IT oraz sędziowaniu piłkarskim. Często opierałem się na pamięci zamiast na realnych danych.

## Funkcjonalności (MVP)

System umożliwia:
- definiowanie umiejętności w określonych domenach,
- ocenę umiejętności (poziom + refleksja),
- zapisywanie historii ocen,
- analizę postępu w czasie.
